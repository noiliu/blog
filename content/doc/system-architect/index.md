##### 线性规划
- 在一组约束条件下来寻找目标函数的极值(极大值和极小值)问题。
- 线性规划问题的数学模型通常由线性目标函数、线性约束条件、变量非负条件组成(实际问题中的变量一般都是非负的)。
- 线性规划问题就是面向实际应用，求解一组非负变量，使其满足给定的一组线性约束条件，并使某个线性目标函数达到极值。满足这些约束条件的非负变量组的集合称为可行解域。可行解域中使目标函数达到极值的解称为最优解。
- 线性规划问题的最优解要么是0个(没有)，要么是唯一的(1个)，要么有无穷个(只要有2个，就会有无穷个)。
- 在实际应用中，可以直接求约束条件方程组的解，即是交叉点，将这些解代入到目标函数中判断是否极值即可。

![Pasted image 20260512141858.png](/static/images/Pasted%20image%2020260512141858.png)

解：列出不等式及目标函数
- 假设生成x吨产品I，y吨产品II，则根据图中信息可知约束条件有：1. 甲材料有4吨，即x+y<=4，2. 乙材料有12吨，即4x+3y<=12，3. 丙材料有6吨，即x+3y<=6，4. 变量非负条件，即x>=0, y>=0，目标函数为z=9x+12y，最大利润即为求表达式z的最大值。
- 两个未知数，两个方程可以求出未知数，故将三个条件分别两两一组联立求解，如果没解就算了（不一定都有解），如果有解就用得出的解代入第三个条件，看是否都能满足，都能满足的解即为最终结果，若存在多个解，则将解分别代入目标函数，题中若问最大值，则代入得出的结果最大的一组解即为最终结果，若题中问最小值，则代入得出的结果最小的一组解即为最终结果。
- 1与2联立求解得：x=0，y=4，代入3：x+3y=0+3 * 4=12，不满足<=6的条件
- 1与3联立求解得：x=3，y=1，代入2：4x+3y=4 * 3+3 * 1=15，不满足<=12的条件
- 2与3联立求解得：x=2，y=4/3，代入1：2+4/3=10/3，满足<=4
- 故当x=2，y=4/3时会产生最大利润，即9x+12y=9 * 2+12 * 4/3=34万元

##### 伏格尔法

![Pasted image 20260512151032.png](/static/images/Pasted%20image%2020260512151032.png)

解：
1. 先算出每行和每列分别计算出最小的和次小的差值，
2. 从行中和列中选出一个差值最大的，在对应选中的最大的行或列中找到最小的，优先供给最小的，计算运输成本
3. 如果某一行某一列供应量和需求量都变成0了，就可以把这一行这一列都划掉了
4. 重复以上步骤，将所有运输成本相加即为最终结果

- 第一行最小的是1，次小的是5，差值为4
- 第二行最小的是2，次小的是3，差值为1
- 第三行最小的是3，次小的是6，差值为3
- 第一列最小的是2，次小的是3，差值为1
- 第二列最小的是1，次小的是4，差值为3
- 第三列最小的是3，次小的是6，差值为3
- 行是4,1,3，列是1,3,3，所以差值最大的是第一行4，所以从第一行找到最小的就是1，即煤场A优先供给工厂Y，A最多可以供12万吨，Y需要12万吨，所以A全部供给Y了，运输成本为12 * 1
- 此时第一行煤场A的供应量为0，第二列工厂Y的需求量为0，划掉第一行，划掉第二列，重新计算剩余行和列的最小和次小差值即重复以上步骤

- 还剩余第二行，第三行，第一列，第三列
- 第二行最小的是3，次小的是2，差值为1
- 第三行最小的是3，次小的是7，差值为4
- 第一列最小的是2，次小的是3，差值为1
- 第三列最小的是3，次小的是7，差值为4
- 行是1,4，列是1,4，此时行或列的差值最大的是4，且存在第三行的差值是4，第三列的差值也是4，则继续找
- 分别从差值最大的第三行和第三列中找最小的，第三行是3,7，最小的是3，第三列是3,7，最小的是3，这时候差值最大的和数值最小的都相同，则可以任取一行或者任取一列
- 假设取工厂Z这一列，那么这一列中找到最小的就是3，优先供给最小的，即煤场B供给工厂Z，煤场B最多可以供应14万吨，工厂Z需要13万吨，则B供给Z13万吨，B剩余1万吨，Z全部满足，需求量为0，此时运输成本为13 * 3
- 此时工厂Z的需求量为0，划掉第三列，煤场B还剩余1万吨，所以第二行还存在，且供应量变成1

- 此时只剩余第一列，且第一列只存在第二行和第三行了
- 直接选最小的即煤场B优先供给工厂X，煤场B只有1万吨了，供给X，运输成本为1 * 2，工厂X还需要10万吨，煤场C供给X，运输成本为10 * 3

- 故总的运输成本为12 * 1+13 * 3+1 * 2+10 * 3=12+39+2+30=83

##### 状态转移矩阵

![Pasted image 20260512182231.png](/static/images/Pasted%20image%2020260512182231.png)

解：
1. 当前市场占有率各为50%，也就是说当前A：0.5，B：0.5
2. 一个月后，
	1. A的0.5中有0.8即80%还是自己的，另外0.2即20%转移给B了
	2. B的0.5中有0.6即60%还是自己的，另外0.4即40%转移给A了
	3. 综上，此时A：A自己剩余的+B转移给A的=0.5 * 0.8+0.5 * 0.4 = 0.6
	4. 综上，此时B：B自己剩余的+A转移给B的=0.5 * 0.6+0.5 * 0.2 = 0.4 
	5. （只要算出一个即可，因为就两个产品一个算出来之后，另一个用1减去算出来的即可）
3. 当前A：0.6，B：0.4
4. 两个月后，
	1. A的0.6中有0.8即80%还是自己的，另外0.2即20%转移给B了
	2. B的0.4中有0.6即60%还是自己的，另外0.4即40%转移给A了
	3. 综上，此时A：A自己剩余的+B转移给A的=0.6 * 0.8+0.4 * 0.4 = 0.64
	4. 综上，此时B：B自己剩余的+A转移给B的=0.4 * 0.6+0.6 * 0.2 = 0.36
5. 当前A：0.64，B：0.36
6. 所以：A的份额0.64相对于0.5来说，增加了
